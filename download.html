Ajax Live Search Lite

To install aLs lite; follow the following steps.

  • Login to your admin dashboard.
  • Navigate to your plugins page and click add new.
  • In the search form, type ajax Live Search and click on the result that has picocodes as the author.
  • WordPress will install it for you. Click on activate now to start supercharge your site's search.
